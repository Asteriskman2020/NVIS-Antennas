# NVIS-Antennas

HF antenna designs for Near-Vertical Incidence Skywave (NVIS) regional coverage and DX operation. Each design includes full technical documentation, a high-resolution poster, PDF datasheet, and (where available) an interactive HTML webpage.

### 4. Tri-Band EFHW Inverted-L &mdash; 80m + 40m + 20m

End-Fed Half-Wave antenna covering three bands from a single 39.4 m wire using natural harmonic resonance. 49:1 transformer, inverted-L configuration. NVIS on 80m/40m plus DX on 20m.

## Design Comparison

| Parameter | Balanced Mag Loop | Efficient Fan Dipole | Small-Size Fan Dipole | EFHW Tri-Band |
|-----------|------------------|---------------------|----------------------|---------------|
| Bands | 80m + 40m | 80m + 40m | 80m + 40m | 80m + 40m + 20m |
| Best gain | +1 dBi (40m) | +7.8 dBi (40m) | +5.5 dBi (40m) | +7.0 dBi (20m) |
| Efficiency | 8-55% | 97-99% | 72-86% | 85-96% |
| Footprint | 2 m diameter | 39 x 12 m | 19 x 8 m | 29 x 10 m |
| Cost | $313-$712 | $280-$450 | $80-$150 | $95-$185 |
| Unique feature | Stealth, noise rejection | Maximum performance | Compact size | 20m DX band |
